# GlobalEd Navigator

GlobalEd Navigator is a comprehensive centralized admission portal designed specifically for students in India. It streamlines the complex process of discovering universities, tracking applications, and receiving expert counseling for higher education.

## 🚀 Key Features

- **Centralized Dashboard**: A unified view for students to browse top-tier Indian institutes (IITs, IIMs, AIIMS, etc.) and popular courses.
- **Smart Discovery**: Integrated search and filter system by Level (UG/PG) and Stream (Engineering, Management, Medical, etc.).
- **Application Tracking**: Real-time progress monitoring for university applications, from document verification to final enrollment.
- **Expert Counseling**: 1-on-1 booking system to connect with education advisors for personalized guidance.
- **Mobile-First Experience**: Fully optimized responsive design that feels like a native app on mobile while offering a robust desktop experience.
- **Secure Profile Management**: Personal data, educational history, and document storage tailored to Indian regulatory requirements (e.g., Aadhar verification).

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Integration**: [Genkit](https://firebase.google.com/docs/genkit) (Google Gemini)
- **Database/Auth**: Simulated state with local storage (Ready for Firebase integration)

## 📂 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (ShadCN) and navigation elements.
- `src/ai`: Genkit flows and AI prompt definitions.
- `src/hooks`: Custom React hooks for state and UI management.
- `src/lib`: Utility functions and placeholder data.
