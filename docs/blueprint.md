# **App Name**: GlobalEd Navigator

## Core Features:

- User Authentication: Enable secure login and registration flows with input validation for email and password fields.
- Intuitive Home Dashboard: Display a personalized greeting, quick search, featured universities, and popular courses with a bottom navigation system.
- University & Course Catalog: Browse a comprehensive listing of universities in card format, including search functionality and filtering options.
- Detailed University Profile: View extensive university information across different tabs such as overview, courses, and admission requirements.
- Multi-Step Application Form: Guide users through a multi-step application process to input personal, academic details, and conceptual document uploads.
- Application Status Tracking: Monitor the progress of submitted applications via a visual, color-coded stepper timeline, including estimated decision dates.
- AI Application Guidance Tool: Provides a tool within the application flow offering context-aware tips and best practices for optimizing documents and personal statements.

## Style Guidelines:

- Primary color: A professional Soft Blue (#2F80ED), used for key interactive elements, branding, and major headings, signifying trust and clarity.
- Background color: A subtly desaturated Light Grey-Blue (#E5EEF6), derived from the primary hue, providing a clean, calm, and harmonious backdrop to the content.
- Accent color: A vibrant Bright Sky Blue (#66DAFF), selected to draw attention to call-to-action buttons, highlights, and critical information, providing an energetic contrast to the primary.
- Neutral palette: Utilize crisp white for card backgrounds and a dark charcoal for body text to ensure maximum readability and a clean aesthetic, complementing the blue tones.
- Font: 'Inter' (sans-serif) for both headlines and body text, chosen for its modern, objective aesthetic and exceptional legibility across various screen sizes and text densities.
- Employ a set of clean, modern icons, preferably in an outline or filled style with rounded corners, to harmonize with the overall soft and minimal UI design language.
- Modern & Minimal Design: Implement a card-based layout featuring rounded corners and subtle soft shadows, creating a contemporary and user-friendly visual experience across all screens.
- Structured Spacing: Maintain a consistent layout grid with generous 16px padding and uniform margins throughout the application, ensuring visual hierarchy and optimal content readability.
- Subtle & Functional Animations: Integrate smooth page transitions, button ripple effects, splash screen fade, and step-by-step progress animations to enhance user feedback and deliver a polished, responsive feel.
